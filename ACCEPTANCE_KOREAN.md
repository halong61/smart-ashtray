# 한국어 수락 기준

## 1. 기능 수락 기준
- 로그인 화면에서 이메일/비밀번호 입력 후 로그인 가능
- 로그인 성공 시 대시보드로 이동
- 대시보드에서 부스 상태 카드와 부스 목록 표시
- 백엔드 `/api/auth/login`, `/api/auth/me`, `/api/booths`가 정상 응답

## 2. UI/UX 수락 기준
- 주요 텍스트는 한국어로 표시
- 상태 배지와 카드 레이아웃이 기본적으로 확인 가능
- 모바일/데스크톱에서 카드가 자연스럽게 보임

## 3. 운영 기준
- `.env` 설정 후 로컬 실행 가능
- 프론트엔드 빌드 성공
- 백엔드 Python 문법 오류 없음

## 4. 다음 단계 권장
- 실제 JWT 인증 도입
- MQTT 메시지 수신 및 WebSocket 실시간 반영
- TimescaleDB 연동
- Flutter 앱 완성
